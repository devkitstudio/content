# Selection Sort Concept

Selection Sort is a simple and efficient sorting algorithm that works by repeatedly selecting the smallest (or largest) element from the unsorted portion of the list and moving it to the sorted portion of the list.

### How it works:
1. The algorithm maintains two subarrays in a given array:
   - The subarray which is already sorted.
   - Remaining subarray which is unsorted.
2. In every iteration of selection sort, the minimum element (considering ascending order) from the unsorted subarray is picked and moved to the sorted subarray.

### Complexity
- **Time**: O(N²) worst/average/best case. It always searches the entire unsorted array.
- **Space**: O(1)
