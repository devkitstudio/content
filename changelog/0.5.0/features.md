# 0.5.0: The Algorithm Playgrounds Launch 🚀

Welcome to **0.5.0**, a monumental milestone for DevKit Studio! We are thrilled to officially unveil the **Algorithms Module** — a highly interactive, visual laboratory designed to help you deeply understand software engineering patterns across all layers of the stack.

Unlike traditional static tutorials, DevKit Studio now provides real-time, step-by-step execution playgrounds paired with theoretical concepts and practical TypeScript implementations.

## 📚 Comprehensive Knowledge Base
We've launched with a massive initial curriculum covering four distinct domains:

### 1. Frontend Patterns
Complex asynchronous and reactive patterns visualized:
- **Debounce**: See how rapid continuous events are grouped.
- **Throttle**: Understand rate-limiting streams over time.
- **Promise Pool**: Watch massive concurrent fetch requests being tamed by connection pools.

### 2. Database Internals
Peek under the hood of database engines:
- **B-Tree Indexing**: Interactive visualization of page splits, node merging, and tree balancing.
- **LRU Cache**: Watch memory constraint eviction policies in action.
- **Bloom Filter**: Understand false positives and probabilistic hashing.

### 3. Backend & Graphs
Foundational data exploration:
- **Breadth-First Search (BFS)** & **Depth-First Search (DFS)**: Traverse intricate network graphs.
- **Binary Search**: Logarithmic elimination in action.

### 4. The "Classic 6" Sorts
The physical mechanics of array sorting, mapped through Framer Motion layout boundaries:
- **Quick Sort** & **Merge Sort**
- **Bubble Sort** & **Selection Sort**
- **Insertion Sort** & **Heap Sort**

## ⚡ The V2 Interaction Engine
To power these playgrounds, we built an entirely new interactive rendering pipeline:

- **Framer Motion Integration**: Elements mapping to memory don't just disappear; they physically slide, swap, and glow based on their algorithmic state.
- **Execution Log Terminal**: A built-in synced terminal prints the exact logic decisions being made behind the scenes, synced precisely to the UI frames.
- **Playback Control**: Step through arrays and trees at your own pace using the new Speed Configurator (`Slow`, `Normal`, `Fast`, `Max Speed`).
- **Theory & Code Tabs**: Every playground is seamlessly backed by extensive Markdown-driven `Concept` and `Implementation` explainers to connect the theory with the visual outcome.
