> [!TIP]
> Version 0.8.0 focuses heavily on "Observability" — making DevKit Studio smarter, faster, and more reliable by giving our system the ability to autonomously track health and usage data across the entire platform.

### What's New?

#### 📈 Universal Platform Analytics
- **Usage Insights**: We've introduced a comprehensive telemetry system that tracks how individual tools are being used. This helps us prioritize updates for the most popular utilities!
- **Automated Environment Detection**: The studio now automatically adapts and records helpful metrics like Timezones and active Browser configurations, ensuring features remain compatible regardless of where you are working from.

#### 🕵️‍♂️ Automated Crash Reporting
- **Proactive Bug Tracking**: DevKit Studio is now equipped with a robust incident management engine. If a tool crashes or throws an unexpected error on your browser, a secure automated report is immediately pushed to our dashboards so we can patch the issue faster.
- **Native Ad-blocker Bypass**: We implemented a smart tunnel to ensure critical error reports successfully reach our servers even if you are running strict Ad-blocker extensions.

#### 🗃️ Built-in Diagnostics
- Implemented background diagnostic snapshots to continually monitor platform performance without slowing down your active UI workspace.

#### 🎨 "Lost in the Void" 404 Experience
- **Immersive Redesign**: Completely overhauled the "Page Not Found" screen into a lively, interactive space portal.
- **Animated Aesthetics**: Features rotating abstract SVGs, floating developer icons, and a glassmorphism notification card powered by Framer Motion. The new experience seamlessly adapts to both Light and Dark mode preferences.

#### 🚨 "System Overload" Exception UI
- **Redesigned System Crash Screen**: Overhauled the core platform error screen into a premium, interactive interface featuring neon red alert aesthetics, pulsing warning graphics, and a sleek modern diagnostics card.
- **Intelligent Privacy Engine**: In the rare event of a deeply embedded crash, the system now automatically conceals internal infrastructure details from everyday users, ensuring our underlying ecosystem remains 100% secure.

#### ✨ Premium Support Center
- **Immersive Redesign**: Transformed the Support Hub with glassmorphic elements, dynamic ambient lighting that tracks your movement, and seamless native Light/Dark mode transitions.
- **Transparent Open Tracker**: We've launched a new official community repository to empower developers to publicly report bugs, request tools, and shape our future engineering roadmap.
- **Dedicated Resolution Channel**: Introduced a dedicated Intellectual Property & Licensing protocol, ensuring any DMCA or copyright concerns can be securely and amicably resolved directly with our core team.

#### 🧠 Algorithm Studio: Fluid Split Windows
- **Smart Window Focus Integration**: Algorithms Studio now remembers exactly which window pane you are working in (Primary or Secondary). Newly opened algorithms will intelligently spawn into your actively focused pane.
- **Tab Migration Engine**: Rather than opening duplicate instances, if you request an algorithm that is already running in an unfocused window, the engine will instantly "grab" and migrate that interactive tab to your active window. Smooth, intuitive, and designed to minimize clutter.
