### Image Converter

- Convert images between **PNG, JPEG, WebP, AVIF, BMP, GIF** formats
- **Resize controls** with linked width/height (aspect ratio lock)
- **Cover / Contain** fit mode selector
- **Auto center** toggle switch
- Drag & drop file upload
- Side-by-side preview with file size comparison

### JSON Validator & Fixer

- Real-time validation with Monaco editor
- Auto-detect and fix common JSON errors
- Individual fix per error or fix all at once
- Fixed preview with copy and download

### Changelog Page

- Version timeline with date display
- Section tabs for organized release notes
- Markdown content rendering
- State management powered by `@ecosy/store`

### UI Improvements

- Dynamic icon loading
- Footer navigation updated
- Theme switch moved to left header
- **VS Code Marketplace** link in hero section
- **Google AdSense** Auto Ads integration
